# QA Checklist

Dung checklist nay de test nhanh sau khi da deploy.

## Landing page

- [ ] Trang mo duoc khong loi console nghiem trong.
- [ ] Anh hero/about/packages load duoc.
- [ ] Section feedback hien anh.
- [ ] Feedback co nut next/prev hoat dong.
- [ ] Package cards khong bi tran chu desktop.
- [ ] Package cards khong bi tran chu mobile.
- [ ] Nut CTA scroll dung ve form dat lich.
- [ ] Music toggle bat/tat duoc neu dung nhac nen.

## Admin

- [ ] Dang nhap thanh cong.
- [ ] Sai mat khau co bao loi.
- [ ] Sua noi dung landing va luu thanh cong.
- [ ] Refresh landing thay doi noi dung.
- [ ] Them goi moi thanh cong.
- [ ] Sua goi thanh cong.
- [ ] Xoa goi thanh cong.
- [ ] Keo tha thu tu goi thanh cong.
- [ ] Upload anh feedback thanh cong.
- [ ] Xoa anh feedback thanh cong.
- [ ] Cac nut trong admin khong bi tran.

## Packages

- [ ] Goi moi hien trong packages section.
- [ ] Goi moi hien trong form dat lich.
- [ ] Thu tu goi tren landing dung nhu admin.
- [ ] Gia online dung.
- [ ] Gia offline dung.
- [ ] Don vi `/gio` hoac `/buoi` dung o packages section.
- [ ] Don vi dung trong dropdown form.
- [ ] Don vi dung trong tom tat form.
- [ ] Don vi dung trong modal thanh toan.

## Booking

- [ ] Form bat loi khi thieu ten.
- [ ] Form bat loi khi ngay sinh sai.
- [ ] Form bat loi khi thieu phone/email.
- [ ] Chon hinh thuc thi dropdown goi cap nhat.
- [ ] Tom tat gia hien ngay sau khi chon goi.
- [ ] Lich hien 21 ngay.
- [ ] Slot da dat khong hien lai.
- [ ] Chon slot xong nut thanh toan sang trang thai bam duoc.
- [ ] QR thanh toan co dung so tien.
- [ ] Noi dung CK dung ma goi + phone.
- [ ] Bam xac nhan thanh toan luu Sheet.
- [ ] Tao Calendar event.
- [ ] Gui email khach.
- [ ] Gui email chu trang.

## Error handling

- [ ] Neu Sheet/API loi, user thay thong bao de hieu.
- [ ] Loi client duoc ghi vao `Error log`.
- [ ] Loi server duoc ghi vao `Error log`.
- [ ] Spam submit nhieu lan bi chan tam thoi.
- [ ] Email fail van luu booking va ghi `Email log`.

## Mobile

- [ ] Header khong che noi dung.
- [ ] Menu mobile mo/dong duoc.
- [ ] Package carousel/scroll khong gay tran ngang bat thuong.
- [ ] Form input de bam.
- [ ] Modal lich vua man hinh.
- [ ] Modal thanh toan khong bi cat thong tin quan trong.
